### 에러 - 이미 사용 중인 포트

* 에러

```
`bind': Address already in use - bind(2) for 127.0.0.1:4000 (Errno::EADDRINUSE)
```

* 해결

해당 포트를 사용중인 프로세스의 PID 확인 후, 종료

```
lsof -iTCP -n -P | grep {포트번호}
kill -9 {PID}
```
